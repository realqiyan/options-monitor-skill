# 期权策略监控

## 工具职责

1. **数据聚合** - 获取所有策略、持仓、价格数据
2. **止损告警** - 检查并输出止损触发状态
3. **策略规则分析** - 基于策略规则检查Delta、到期日、持仓调整需求

## 安装

```bash
cd scripts && npm install && npm link
```

## 使用方式

```bash
options-monitor
```

## 输出格式

监控结果保存在 `scripts/output/latest-report.txt`，为可读的文本格式，与控制台输出一致。

### 报告内容

- **止损告警** - 触发止损的期权持仓（如有）
- **持仓调整建议** - 基于策略规则的调整建议（如有）：
  - Delta调整建议
  - 到期日警告
  - 卖Put收益平仓建议（Wheel策略，收益>=80%）
  - 被指派风险警告（价内卖Put）
- **策略状态** - 每个策略的详细信息：
  - 策略ID、名称、代码
  - 标的股票代码和价格
  - Delta分析（基于策略规则的建议）
  - Theta
  - 持股数量、手数、期权持仓数
  - 累计收益（期权收益 + 总收益）
  - 期权持仓详情：代码、方向（买/卖）、类型（Call/Put）、合约数、DTE、行权价、成本价、现价
- **无期权持仓的策略** - 需要关注是否开仓
- **错误信息** - 数据获取失败（如有）

## 内置策略规则

### cc_strategy (备兑看涨策略)

- 策略Delta正常区间：0.25-0.75
- 触发调整：Delta <= 0.25 或 >= 0.75，或到期日 <= 3周
- Delta >= 0.75：目标调整至0.5
- Delta <= 0.25：目标调整至max(当前+0.1, 0.35)
- 到期 <= 3周：Delta向0.5靠近0.1

### wheel_strategy (车轮策略)

- 开仓卖Put Delta范围：0.10-0.35
- 卖Put收益 >= 80%：建议平仓
- 卖Put价内(ITM)：被指派风险警告
- 持有股票时：卖Call策略

### default (默认卖期权策略)

- Delta范围：0.15-0.35
- 优先周期权
